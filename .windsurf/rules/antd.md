# Ant Design (antd)

- Prefer `antd` components for common UI patterns.
- Use `antd` layout/spacing patterns (e.g., `Space`, `Flex`, `Row`/`Col`) over ad-hoc CSS.
- For detailed component APIs, use `docs/antd/llms.txt` and `docs/antd/llms-full.txt` for comprehensive documentation.
