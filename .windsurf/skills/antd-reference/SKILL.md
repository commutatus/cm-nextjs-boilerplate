## Ant Design Reference

- Prefer `antd` components for common UI patterns.
- Reference `llms.txt` in this skill folder for component APIs and theming guidance.
- Only reference `llms-full.txt` if you need deeper component API details.
- Use `antd` layout/spacing patterns (e.g., `Space`, `Flex`, `Row`/`Col`) over ad-hoc CSS.
